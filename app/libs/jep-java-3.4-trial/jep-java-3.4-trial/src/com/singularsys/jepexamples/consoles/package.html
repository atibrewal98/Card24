Examples that allow console interaction with the Jep parsers using a command line interface. The {@link com.singularsys.jepexamples.consoles.Console} class provides a framework for implementing interactive applications and applets. It also serves as a interactive command line for the standard configuration of Jep. {@link com.singularsys.jepexamples.consoles.CPConsole} is a interactive command line for the standard configurable parser.
