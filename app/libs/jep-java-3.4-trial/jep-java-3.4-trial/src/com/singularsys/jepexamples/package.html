Container package for all examples.
  • {@link com.singularsys.jepexamples.consoles} Contain a set of console applications for interactive sessions using different configurations.
  • {@link com.singularsys.jepexamples.applets} contains some graphical applets using jep.
  • {@link com.singularsys.jepexamples.diagnostics} contains various applications for testing the performance of jep.
